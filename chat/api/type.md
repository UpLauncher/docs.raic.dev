# 型 (Type)

型 (例: `<MessageContent>`) についての解説

## Rai Chat

Rai Chat で使用される型

### MessageContent

ベースの型は string で、メッセージを送信/返信する内容です。

### MessageId

ベースの型は string (数字)で、返信先のメッセージの ID を指定します。

### UserId

ベースの型は string で、ユーザーの ID を指定します。
